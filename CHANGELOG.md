# Changelog

All notable changes to FlashVLM will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added

- Initial release of FlashVLM
- Support for LLaVA, Qwen-VL, InternVL, and Phi-Vision architectures
- Vision encoders: CLIP, SigLIP, DINOv2
- Projection layers: MLP, Q-Former, Cross-Attention
- Task modules: VQA, Captioning, Grounding, OCR, Reasoning
- Training strategies: SFT, DPO, RLHF
- LoRA/QLoRA fine-tuning support
- Generation utilities: beam search, top-k/top-p sampling, streaming
- Solutions: Multimodal Chat, Document QA, Image Analyzer
- Benchmarking suite with standard VLM metrics
- CLI interface for inference and training
- Docker support for deployment
- Comprehensive documentation and examples
